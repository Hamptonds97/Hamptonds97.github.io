Computer Graphics Assignment

For this assignment I was given image slices of a CT scan of a persons head. The slices we of the top, front and side of the head. The task was to take these slices and create a GUI that can display them and allow the user to interact with them.

I gained a mark of 62% for my assignment.

The first task I had with this assignment was to display 3 images that each contained all slices of that image. For the front and side images there were 255 slices and the top image had 155. I decided to implement a slider function that would allow users to go through each slice, quickly and easily. The sliders are very pleasing and responsive to use. They make interacting with the system pleasant and look pleasant too.

Once this was implemented and worked correctly I decided to implement a button that, when clicked, would allow for a Max Intensity Projection(MIP) of the image to be displayed. Each part of the image had its own button for MIP. The screenshot image at the top of this page, shows what the images look like when MIP has been added. MIP is a method that consists of projecting the voxel with the highest attenuation value on every view throughout the volume onto a 2D image.

This is useful in reading CT scans as the method tends to display bone and contrast material-filled structures preferentially, and other lower-attenuation structures are not well visualised. MIP is primarily used to improve the detection of pulmonary nodules asses their profusion. Another use for MIP is that sections of variable thickness are excellent for assessing the size and location of vessels, including the pulmonary arteries and vein.

After implementing MIP I decided to add a feature that would resize the images. To do this I used two sliders the left slider controls the length of the image while the right slider controls its width. To change the size of the image all the user needs to do is adjust the sliders and then click on the button to change the size of an image. Each resize button uses the same sliders therefore the button must be clicked so that the system knows which image to resize. In this system I decided to use the nearest neighbour approach to resizing.

Nearest-Neighbour interpolation is a simple method of multivariate interpolation in one or more dimensions. Interpolation is the problem of approximating the value of a function for a non-given point in some space when given the value of that function in points around that point. This means that the nearest neighbour algorithm selects the value of the nearest point and does not consider the values of neighbouring points at all. This allows the image to be resized, though creates sharp and blocky edges when the image is bigger. There are other algorithms that can correct this such as bilinear interpolation and trilinear interpolation. These help smooth the lines in the image, preserving the resolution in the image when it is made larger.

This assignment was a great introduction into computer graphics and the methods and ideas studied here helped me a lot with my Final Year Project work, helping me understand how images are processed as this is vital to the preprocessing stages in that project. I was very happy with my submission, I believe the system was good because of its functionality but also its design and usability.
